# resume-qa
This repository is for course assignments, test assignments, and showing the qa knowledge level

Меня зовут Лифантьева Алина, в этом репозитории я размещаю решения заданий с курсов и мои идеи использования инструментов QA в улучшении качества продукта через комплексное тестирование. Я собираю наборы по запрсам API, консольным bash-решениям для автоматизации, а также работаю с тестовыми артефактами (баг-репорты, тест-кейсы, тест-планы). 

## Pet-Projects

### Performance Testing Lab
[Репозиторий с нагрузочным тестированием](https://github.com/ralinka17/performance-testing-lab)
- JMeter с созданием тестовых планов для HTTP/WebSocket с параметризацей данных (использование входных данных из внешних источников) через CSV-файлы
- k6 javascript для smoke-тестирования (быстрая проверка) и нагрузочного тестирования  (определение пределов производительности)
- Автоматизация CI/CD (GitHub Actions), конфигурация k6-пайплайнов для регулярного запуска тестов
- План тестирования, анализ метрик (response time, RPS, error rate) и отчеты с выводами по тестированию
